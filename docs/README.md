# Handbooks

User-facing guides for each role.

- [**ADMIN.md**](ADMIN.md) — for school administrators. Full system control.
- [**TEACHER.md**](TEACHER.md) — for teachers. Grading, attendance, reports.
- [**STUDENT.md**](STUDENT.md) — for students. Viewing progress, downloading reports.

Each handbook is also accessible from inside the app via the **Help** entry in
the sidebar.

For deployment & developer docs, see the top-level `DEPLOY.md` and `README.md`.
